# Soukly

This folder now contains a working English MVP front-end for a UAE-focused second-hand marketplace.

Current scope:
- Sign in flow
- Add product flow
- Product image upload with browser preview
- Instagram-style hashtag support on product creation
- Manual brand entry
- Shopping-style chip filters for brand, category, location, condition, and delivery
- Price range filtering
- Product detail view
- Favorites flow
- Favorites badge in header
- Make offer flow
- Seller offer accept/reject controls
- Product detail comments
- Comment likes
- Product-based buyer/seller messaging
- Notifications badge for pending offers and unread messages
- Seller profile page
- Marketplace sample listings
- Browser-based local storage so the demo keeps data after refresh

How to use:
1. Open `index.html` in a browser
2. Sign in from the home screen
3. Go to `Add product`
4. Publish a listing
5. Open `My profile` to see the storefront update

Important note:
- This is still a front-end MVP, not a production app
- Login is a demo flow stored in the browser
- There is no real database, payment, or image upload yet

Suggested next step:
